# crawler-demo

---

## 技术栈

nodejs + nestjs + playwright + typescript

## Development Setup

```bash
# 构建
$ yarn build

# 启动
$ yarn start

# 代码格式化
$ yarn format

# tslint
$ yarn lint

# 单元测试
$ yarn test:unit

# e2e测试
$ yarn test:e2e

# 部署serverless
$ yarn sls:deploy

```
